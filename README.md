# 前言

随着移动互联网的快速发展，远程在线诊疗系统成为解决医疗资源分配不均、提高医疗服务效率的有效途径。本项目是基于移动平台的远程在线诊疗系统，采用SSM（Spring、SpringMVC、MyBatis）框架进行开发，旨在为广大患者和医生提供便捷、高效的医疗服务。

# 内容介绍

本项目主要包括以下功能模块：用户注册登录、个人信息管理、在线问诊、医生接诊、药品推荐等。患者可以通过移动端APP进行在线咨询，医生可以实时接诊，根据患者病情开具电子处方。同时，系统还支持药品推荐，方便患者购买。

# 技术介绍

## 语言：Java

## 使用框架：Spring、SpringMVC，MyBatis，微信小程序

## 前端技术：JS、Vue、CSS3，Uniapp

## 开发工具：IDEA/Eclipse，Uniapp

## 数据库：MySQL 5.7/8.0

## 数据库管理工具：phpstudy/Navicat

## JDK版本：jdk1.8

## Maven：apache-maven 3.8.1-bin

## 前端环境：Node.Js 12\14\16
